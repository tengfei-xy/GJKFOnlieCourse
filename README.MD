# 国家开发大学命令行网课

## 项目说明

该项目是一种通过命令行来学习网课的方式。

> 1. Windows不支持使用，除非在windows安装bash/zsh的执行环境
> 2. macOS/Linux需要安装curl和jq命令
> 3. 对脚本使用用疑问，欢迎交流，联系方式位于[个人首页](https://github.com/tengfei-xy)

## learning.sh - 学习脚本

**变量说明：**

将脚本下载后需要修改的变量是

**header_cookie_index**

> 作用: 课程列表页的cookie
> 链接: https://menhu.pt.ouchn.cn/site/ouchnPc/index
> 格式: "Cookie: eai-sess=xxxxxxxxx; UUkey=xxxxxxxxxx"



**header_cookie**

> 作用: 具体课程的cookie
> 链接: https://lms.ouchn.cn/course/xxxxxxxxxxx/ng#/
> 格式: "Cookie: session=V2-60000000002-xxxxxxxx; HWWAFSESID=xxxxxx; HWWAFSESTIME=xxxxx"



**脚本思路：**

1. 对于参考资料、页面、视频的完成都是通过脚本汇总的`enter_learn_xxxxxx()`函数实现

---

**注意：**

1. 形考作业、问卷调查未完成
1. 词汇表的后台系统似乎有问题暂时无法完成
